# UGK-Server

多人联网游戏demo，UGK-Server：unity、go、kcp server 。开发中......

## 服务

| 服务	        | 描述      |
|------------|---------|
| ugk-common | 公共逻辑封装  |
| ugk-game   | 游戏      |
| ugk-gate   | 网关，消息转换 |
| ugk-lobby  | 大厅，一般逻辑 |



## 技术选择
* Unity 客户端
* go 服务器
* kcp 网络通信
